FENZI is a GPU molecular dynamics code designed for high-performance
simulation of large biomolecular systems. FENZI is distributed free
of charge and includes source code.

All distributions of FENZI include the following files:

 announce.txt
        This file contains the announcement for this version of FENZI.

 notes.txt
        This file includes directions for compiling and running FENZI
        and advice for resolving problems and reporting bugs.

 license.txt
        This file contains the license under which FENZI is distributed
        and to which you must agree in order to use the program.

 manual.txt
	This file contains .............

Full documentation for using FENZI is available from the FENZI web site:
        http://code.google.com/p/fen-zi/

FENZI is an effort of Taufer's group (Global Computing Lab) and
Patel's group () at the University of Delaware. The sotware is
developed and freely distributed (with source code) for molecular
dynamics studies in structural biology.  For more information, see
http://code.google.com/p/fen-zi/
